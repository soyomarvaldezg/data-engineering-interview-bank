# Cloud Platforms para Data Engineering

## ¿Por qué Cloud?

Data Engineering en 2025 = Cloud. No es opcional. Las 3 main platforms:
- **AWS**: Market leader, widest ecosystem
- **GCP**: Strong for analytics (BigQuery), ML (Vertex AI)
- **Azure**: Enterprise (Microsoft integration), strong governance

## Temas Cubiertos

1. **AWS Data Stack** — S3, EC2, EMR, Lambda, Redshift, Glue
2. **GCP Data Stack** — BigQuery, Dataflow, Cloud Storage, Pub/Sub
3. **Azure Data Stack** — Data Factory, Synapse, Blob Storage, Stream Analytics
4. **Multi-Cloud Strategy** — When to use each, trade-offs

## Recomendación de Estudio

**Para Mid devs:**
- Aprende AWS (most common)
- Understand trade-offs vs GCP/Azure

**For Seniors:**
- Domina 1 platform deeply
- Entiende trade-offs entre los 3
- Sabe cuándo migrate/switch
