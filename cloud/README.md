# Cloud Platforms para Data Engineering

## ¿Por qué Cloud?

Cloud no es opcional. Las 3 plataformas principales:

- **AWS**: Market leader, mayor ecosistema
- **GCP**: Fuerte en analytics, en DWH (BigQuery), ML (Vertex AI)
- **Azure**: Enterprise (Microsoft integration)

## Temas Cubiertos

1.  **AWS Data Stack** — S3, EC2, EMR, Lambda, Redshift, Glue
2.  **GCP Data Stack** — BigQuery, Dataflow, Cloud Storage, Pub/Sub
3.  **Azure Data Stack** — Data Factory, Synapse, Blob Storage, Stream Analytics
4.  **Multi-Cloud Strategy** — When to use each, trade-offs

## Recomendación de Estudio

**Para desarrolladores de nivel intermedio (Mid devs):**

- Aprende AWS (la más común)
- Entiende los trade-offs vs GCP/Azure

**Para Seniors:**

- Domina a fondo una plataforma
- Entiende los trade-offs entre las 3
- Sabe cuándo hacer migrate/switch
