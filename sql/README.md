# SQL en Entrevistas de Data Engineering

## ¿Por qué SQL es importante?

SQL es la **habilidad número 1** en data engineering.  
En entrevistas, esperan que domines:

- Window Functions
- Joins complejos
- CTEs y subqueries
- Query optimization
- Índices y performance

## Temas cubiertos

1. Window Functions (ROW_NUMBER, RANK, LAG/LEAD, etc.)
2. Joins & Subqueries
3. CTEs & Aggregations
4. Query Optimization

## Recomendación de estudio

Comienza con Window Functions → luego Joins → después Optimization.
